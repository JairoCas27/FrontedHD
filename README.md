# 🚗 Urban Park Frontend

Este proyecto es el **frontend** de **Urban Park**, una plataforma web desarrollada con **React** enfocada en la **gestión inteligente de estacionamientos (Smart Parking)**.  
Permite digitalizar, automatizar y optimizar el control de parqueos en entornos residenciales, corporativos y urbanos.

Incluye un diseño **moderno y responsivo**, navegación fluida tipo **SPA**, y una arquitectura escalable basada en componentes reutilizables.

## ✨ Características

* 🚗 **Gestión inteligente de estacionamientos (Smart Parking)**
* 📱 **Diseño responsivo** adaptable a móviles, tablets y escritorio
* ⚡ **Arquitectura SPA** con navegación dinámica
* 🧩 **Componentes reutilizables** (Hero, Secciones con imagen, Layout)
* 🧠 **Plataforma SaaS escalable** para múltiples usuarios y condominios
* 🔐 **Control de accesos y monitoreo en tiempo real**
* 🎨 **Interfaz moderna y amigable** centrada en la experiencia del usuario

## ⚙️ Dependencias

Definidas en `package.json`:

### 🧩 Principales

| Paquete | Descripción |
|---|---|
| **react** | Librería principal para la construcción de interfaces |
| **react-dom** | Renderizado en el navegador |
| **react-router-dom** | Manejo de rutas y navegación SPA |
| **react-bootstrap** | Componentes UI basados en Bootstrap |
| **bootstrap** | Framework de estilos |
| **react-icons** | Librería de íconos |
| **lucide-react** | Íconos SVG modernos |
| **framer-motion** | Animaciones fluidas |
| **react-toastify** | Notificaciones visuales |

## 🛠️ Configuración

1. Clonar el repositorio:

```bash
git clone <repo-url>
```

2. Instalar dependencias:

```bash
npm install
```

***

## ▶️ Ejecución

Para iniciar el proyecto en modo desarrollo:

```bash
npm run dev
```

Luego abre en tu navegador:

```
http://localhost:5173
```

* 🅿️ Gestión de espacios de estacionamiento en tiempo real
